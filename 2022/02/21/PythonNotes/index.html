环境搭建

pip:包管理

  1. 查看已经安装的包:pip list
  2. 安装包:pip install 包名
  3. 卸载包:pip uninstall 包名
  4. 下载到本地再安装
  5. 升级:pip install --upgrade 包名
  6. 查看说明文档:pip help

基础语法

变量

  • 是什么

    变量是一个容器，是内存中的一段空间，是内存中的某个位置
    变量是一个“名字”，“标签”，用来区分另一个值
  • 怎么用
    • 定义
      a = 100
b = "kobe"
c = a + 4
d = b
    • 输出
      # 直接输出
print('kobe')
# 变量输出
a = 'kobe'
print(a)
  • 命名规则
    1. 字母数字下划线，数字不开头
    2. 不包含空格
    3. 不与函数/关键字名称相同(print)
    4. 简单易懂，见名知意

数据类型

  1. 数值型：整数、浮点数
  2. 字符型
  3. 布尔类型：真假、假范围
  4. 类型转换：数值–>字符、字符–>数值

字符串

  • 常用的转义字符

    # a = 'I'm a teacher' 会出现无法识别的错误
a1 = "I'm a teacher"
a2 = 'I\'m a teacher'
# 用'\'连接上一行和下一行
s1 = 'kobe\
is a mentor'
# 用'\n'换行
s2 = "kobe\n is a mentor"

  • 基本运算

    1. 计算长度:len('kobe')
    2. 截取(切片):string(开始:结束:步长)

运算符

  • 类型转换
    a = "8"
b = "3"
c = a + b
print("a+b=", c)
print("type(a)=", type(a))
print("type(b)=", type(b))
ia = int(a)
ib = int(b)
ic = ia + ib
print('ia+ib=', ic)
print('type(ia)=', type(ia))
print('type(ib)=', type(ib))

a = '圆周率是：'
b = '3.14'
# print(a+b),会报错，整型不能与字符串进行计算
print(a + str(b))
print(a + repr(b))
  • 输入输出(格式化)
    1. print()
      n = 3
s = '土豆'
print('我买了', n, '斤', s)
print('我买了%s斤%s' %(n, s))
print('我买了{}斤{}'.format(n, s))
# 这里的%S和{}其实就是占位符
    2. input()
      language = input('输入语言名称:')
print('您输入的语言名称是:', language)

a = input("请输入一个数值：")
print(a + 2) # 此语句会报错，python中会将输入的值当做字符串处理，所以字符串不能与数值型变量进行加减运算
print(int(a) + 2)

f = input('请输入一个整数:')
print(f * 3) # 字符的重复输出

数据结构

列表(==可变，有序， 允许重复==)

  • 定义
    a = [1, 3, 4, 5]
a1 = [1, '哈哈哈', [1, 3, '哈哈哈']]
b = [] # 构建一个空列表
  • 访问:a[start:end:step] (区间为左闭右开)
    # 索引访问，这里注意正索引从0开始，负索引从-1开始
a = [1, 2, 3, 4, 5, 6, 7, 8, 9, 0]
print(a[0:4:2])
print(a[0:3]) # 默认步长为1
print(a[:3]) # 默认左边从0开始，步长为1
  • 添加（a.append()）
    a = [1, 2, 3, 4, 5, 6, 7, 8, 9]
a.append(5) # 在列表末尾添加元素整数5
a.append('5') # 在列表末尾添加元素字符串"5"
  • 删除(删除列表中的元素、删除列表)
    a = [1, 2, 3, 4, 5, 6, 7, 8, 9]
del a[2] # 删除第三个元素3
del a  # 删除列表a

元组(==不可变， 有序，允许重复==)

  • 创建
    • 基本语法:a = (4, 5, 6)
    • 与列表的区别
    • 注意：元组内如果只有一个值，要在最后加上逗号:a = (4, )
    • 具体实现
      a = (4, 5, 6, 4, 9, 0, 2)
print("a=", a)

b = (4)
c = (4, )
print("b=", b)
print("type(b)=", type(b))
print("c=", c)
print("type(c)=", type(c))
  • 访问
    • 访问方式与列表一致
      a = (3, 4, 5, 6, 7, 8, 9, 0)
print(a[3])
print(a[-2])
print(a[2:5])
print(a[2:6:2])
print(a[:6])
print(a[3:])
print(a[5:-1])
  • 操作
    • 删除:直接使用del,因为元组不能修改，所以所谓元组的删除操作是指将整个元组删除
      # 删除整个元组
a = (3, 54, 6)
del a
    • 拼接
      a = (3, 4, 5)
b = (3,)
c = a + b
print(c)
    • 计算长度
      a = (3, 54, 6)
print(len (a))
    • 迭代
      a = (3, 4, 5)
b = a * 3
print(b)

字典(==可变，无序，键不重复==)

  • 定义:键值对(key:value)组成
    d = {
    "小明": 18,
    "小兰": 17,
    "小刚": 16
}
print("d=", d)
print('type(d):', type(d))
  • 引用

    • 引用a["小明"]

      因为字典是无序的，所以也无法通过索引来引用字典

    • 修改a[小明] = 14

  • 增加(如果用来赋值的键在原字典中不存在，则直接为添加操作；如果用来赋值的键在原字典中存在，则相当于修改操作)
    • 语法:a['小红'] = 21
  • 删除
    • 语法
      # 定义一个字典
d = {
    "小明": 18,
    "小兰": 17,
    "小刚": 16
}
# 删除某一个键值对
del d["小明"]
# 删除整个字典
del d

集合（==无序，不重复==）

  • 定义
    # 第一种构建方式：直接构建一个集合
mySet = {"小明", "python教程", "基础知识"}
# 第二种构建方式：类型转换（将一个列表/元组转换为集合
mySet = set(["小明", "小红", "小刚"])
mySet2 = set(("小明", "小红", "小刚" ))
  • 添加:mySet.add("小兰")
  • 删除
    • 删除集合中的元素
      # pop()方法：随机删除一个元素（如果集合中都是字符串，最有可能删除最小值，如果是字符串，则为随机删除）
mySet.pop()
# remove()方法：删除指定的元素
mySet.remove("小明")
del mySet("小明")
# clear()方法：删除集合中全部元素
mySet.clear()
    • 删除集合:del mySet
  • 运算
    • 交集(&):mySet & mySet1
    • 并集(|):mySet | mySet1
    • 差集(-):mySet - mySet1

列表、元组、字典和集合的对比

数据结构 是否可变 是否重复 是否有序 定衣符号
列表：list 可变 可重复 有序 []
元组：tuple 不可变 可重复 有序 ()
字典：dictionary 可变 键不重复 无序 {key:value}
集合：set 可变 不可重复 无序 {}
