pip:包管理

  1. 查看已经安装的包:pip list
  2. 安装包:pip install 包名
  3. 卸载包:pip uninstall 包名
  4. 下载到本地再安装
  5. 升级:pip install --upgrade 包名
  6. 查看说明文档:pip help

      变量

  • 是什么

    变量是一个容器，是内存中的一段空间，是内存中的某个位置
    变量是一个“名字”，“标签”，用来区分另一个值
  • 怎么用
    • 定义
    • 输出
  • 命名规则
    1. 字母数字下划线，数字不开头
    2. 不包含空格
    3. 不与函数/关键字名称相同(print)
    4. 简单易懂，见名知意

数据结构

列表(==可变，有序， 允许重复==)
  • 定义
  • 访问:a[start:end:step] (区间为左闭右开)
  • 添加（a.append()）
  • 删除(删除列表中的元素、删除列表)

元组(==不可变， 有序，允许重复==)

  • 创建
    • 基本语法:a = (4, 5, 6)
    • 与列表的区别
    • 注意：元组内如果只有一个值，要在最后加上逗号:a = (4, )
    • 具体实现
  • 访问
    • 访问方式与列表一致
  • 操作
    • 删除:直接使用del,因为元组不能修改，所以所谓元组的删除操作是指将整个元组删除
    • 拼接
    • 计算长度
    • 迭代

字典(==可变，无序，键不重复==)

  • 定义:键值对(key:value)组成
  • 引用
    • 引用a["小明"]

      因为字典是无序的，所以也无法通过索引来引用字典

    • 修改a[小明] = 14
  • 增加(如果用来赋值的键在原字典中不存在，则直接为添加操作；如果用来赋值的键在原字典中存在，则相当于修改操作)
    • 语法:a['小红'] = 21
  • 删除
    • 语法

集合（==无序，不重复==）

  • 定义
  • 添加:mySet.add("小兰")
  • 删除
    • 删除集合中的元素
    • 删除集合:del mySet
  • 运算
    • 交集(&):mySet & mySet1
    • 并集(|):mySet | mySet1
    • 差集(-):mySet - mySet1

列表、元组、字典和集合的对比

数据结构 是否可变 是否重复 是否有序 定衣符号
列表：list 可变 可重复 有序 []
元组：tuple 不可变 可重复 有序 ()
字典：dictionary 可变 键不重复 无序 {key:value}
集合：set 可变 不可重复 无序 {}
